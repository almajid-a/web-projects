# Restaurant Website

## Overview
A multi-page website built using HTML and CSS. It includes a main page and a reservations page with form-based user input, structured layout, and styling.

## Live Demo
https://almajid-a.github.io/web-projects/restaurant-website/

## Features
- Multi-page navigation
- Custom CSS styling
- Reservation form for user input
- Organized file structure

## How to Run

Open `index.html` in a web browser.

## Skills Demonstrated
- HTML structure
- CSS layout and styling
- Form design and user input handling
- Front-end project organization

## Notes
This project focuses on building a structured and interactive front-end experience using core web development technologies.
